# 🚌 DACE Transport Web

**Smart Campus Transport System for Dhaanish Ahmed College of Engineering, Chennai.**

Find bus routes, timings, and boarding points across Chennai — all in one place.

---

## Features

- 🗺️ Browse all college bus routes with boarding points and timings
- 🔍 Search routes by area or stop name
- 🦺 Safety rules & guidelines for students and drivers
- 📬 Feedback and contact form
- 🎨 Animated UI with smooth transitions and mobile-first design

## Tech Stack

- **React + TypeScript** (Vite)
- **TailwindCSS** for styling
- **Framer Motion** for animations
- **React Router** for navigation
- **JSON Server** for mock backend API

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

To run the mock API server:

```bash
npx json-server db.json --port 5000
```
